✅ **Sorting Algorithms**: Bubble, Selection, Insertion, Merge, Quick, Heap Sort  
✅ **Searching Algorithms**: Linear, Binary, Jump, Interpolation, Exponential Search  
✅ **Graph Algorithms**: BFS, DFS, Dijkstra, A*, Prim's, Kruskal's  
✅ **Tree Algorithms**: Binary Tree, BST, AVL Tree operations

## Quick Start

1. **Download** the appropriate file for your OS
2. **Run** the application (no installation needed for Windows)
3. **Select** an algorithm category from the main menu
4. **Choose** a specific algorithm
5. **Adjust** speed and array size
6. **Visualize** and learn!
