# 🎯 Algorithm Visualizer Suite - Quick Start

## 🚀 Launch Application

```bash
cd /Users/swastikparmar/Downloads/visalizer
python3 main_menu.py
```

Or use the launcher:


## ✨ Features

- **Single Window** - No multiple windows, smooth navigation
- **Fast Response** - Instant button clicks (<50ms)
- **25+ Algorithms** - Sorting, Searching, Graph, Tree, Pathfinding, Data Structures
- **Beautiful UI** - Modern dark theme with vibrant colors
- **Back Navigation** - "← Back to Menu" button in every visualizer

## 🎮 How to Use

1. **Launch** - Run the application
2. **Click** - Select any category card
3. **Explore** - Adjust parameters and visualize
4. **Navigate** - Use "← Back to Menu" to return
5. **Repeat** - Try all 6 visualizers!

## 🔧 Troubleshooting

**Blank screen when clicking visualizer?**
- Make sure all `*_embedded.py` files are present
- Check that `utils.py` is in the same directory
- Restart the application

**Slow performance?**
- Adjust the speed slider in each visualizer
- Reduce array size for sorting
- Reduce node count for graphs
